# Weight Object

## Description
This documentation provides information about the `Weight` model, which represents a weight within a workout.

## Model Properties

| Property    | Type            | Description                         |
|-------------|-----------------|-------------------------------------|
| value       | number         | The value of the weight.           |
| displayUnit | [WeightType](../enums/weight.md)      | The display unit of the weight.    |


### Example

```json
{
  "value": 75.5,
  "displayUnit": "kilograms"
}
```
